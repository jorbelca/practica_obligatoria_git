Descripción del curso

En este curso de Git, hemos explorado cómo gestionar versiones de código mediante Git, y la configuración de proyectos git para optimizar el flujo de trabajo con git flow, un conjunto de extensiones que facilita el manejo de ramas en proyectos de desarrollo. También hemos trabajado con submódulos, que permiten incluir y gestionar repositorios dentro de otro, asegurando que los cambios en cada submódulo se reflejen correctamente en los repositorios de origen. Con estos conocimientos, es posible mantener una estructura de proyectos modular y eficiente.
